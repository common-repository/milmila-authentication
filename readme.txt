This is Milmila Affiliate Plugin. All the Affiliates registered in https://affiliate.milmila.com are eligible to list products from https://milmila.com into their website.

Please go through the following link to enable the WooCommerce REST API : https://docs.woocommerce.com/document/woocommerce-rest-api/
Steps to create REST API Key :
1) WordPress permalinks must be enabled at: Settings > Permalinks.
2) Go to: WooCommerce > Settings > Advanced > REST API.
3) Select Add Key. You are taken to the Key Details screen.
4) Add a Description ( Milmila Affiliate REST API ).
5) Select the User you would like to generate a key for in the dropdown.
6) Select a level of access for this API key â€” Read/Write access ( For Milmila Affiliate select Read/Write access ).
7) Select Generate API Key, and WooCommerce creates API keys for that user.
8) Now that keys have been generated, you should see Consumer Key and Consumer Secret keys, a QRCode, and a Revoke API Key button.
9) Copy Customer Key and Customer Secret for future use ( Kindly note that Customer Key and Customer Secret are generated one time and will not be changed untill you create new REST API Key )
10) go to WooCommerce > Settings > Advanced > Legacy API and tick the Enable the legacy REST API checkbox.


In this plugin you are required to check if you are registered as affiliate in https://affiliate.milmila.com, required informations are:
1) Domain name ( Your website domain name)
2) Customer Key
3) Costomer Secret
Now click submit button to know if you are registered as Affiliate in https://affiliate.milmila.com
